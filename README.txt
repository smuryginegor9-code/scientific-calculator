Scientific Calculator (Android, Kotlin, MVVM)

Кратко о задании:
- Android-приложение "Scientific Calculator" на Kotlin.
- Архитектура: MVVM + Navigation Component + Data Binding.
- Сохранение состояния через ViewModel + SavedStateHandle.
- Material 3 (стиль Pixel / Material You) + динамические цвета, если поддерживаются устройством.
- История вычислений через Room.

Реализованные функции:
- Базовые операции: +, -, *, /
- Скобки и приоритет операций
- Проценты: бинарная операция x % y = y*x/100
- Факториал n! (только целые n >= 0, лимит до 20!)
- Квадрат x²
- Степень x^y
- Корень sqrt(x)
- Корень произвольной степени через x^(1/n) (кнопка ⁿ√x вставляет ^(1/)
- Тригонометрия: sin, cos, tan
- Переключатель DEG/RAD
- Логарифмы: ln, log10
- Константы: π = 3.1415926536, e = 2.7182818285
- Ввод десятичных чисел с точкой и запятой (запятая нормализуется в точку)
- DEL, AC, =

Как пользоваться:
- Вводите выражение кнопками, затем нажмите =.
- Примеры:
  - (2+3)*4
  - sin(90)  (в DEG)
  - ln(e)
  - log10(100)
  - 5!
  - 27^(1/3)
  - 50%200  -> 100

Про кнопку %:
- Реализация выбрана как бинарная операция:
  x % y = y * x / 100
- Пример: 50%200 = 100

Режим DEG/RAD:
- DEG: аргументы тригонометрии интерпретируются в градусах.
- RAD: аргументы тригонометрии интерпретируются в радианах.

Обработка ошибок:
- Деление на ноль
- Некорректные/несбалансированные скобки
- sqrt(-1) в вещественных
- ln(x), log10(x) при x <= 0
- tan(x) в точках, где функция не определена
- Факториал для нецелых, отрицательных и слишком больших значений

Запуск:
1. Открыть проект в Android Studio.
2. Дождаться Gradle Sync.
3. Запустить конфигурацию app на эмуляторе или устройстве (minSdk 24).

Тесты:
- Unit-тесты для вычислительного движка находятся в:
  app/src/test/java/com/example/engineeringcalculator/domain/CalculatorEngineTest.kt
- Запуск: ./gradlew testDebugUnitTest
